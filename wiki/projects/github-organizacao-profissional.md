## Lições Aprendidas

- Repositório especial `usuario/usuario` cria o README de perfil público
- `.gitignore` é obrigatório em todo projeto profissional
- Nomes de arquivos/repos com acento, espaço ou maiúsculas são má prática
- GitHub cacheia as páginas — mudanças podem demorar alguns minutos para aparecer
- A API `api.github.com` está bloqueada no ambiente do Claude — renomear repos e editar "About" precisam ser feitos manualmente
