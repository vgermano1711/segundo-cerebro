# Hype Station / ASN Supply — E-commerce

## Objetivo
E-commerce de roupas e tênis com sistema completo de loja, entrega e painel administrativo.

## Status
**Desenvolvido** — múltiplas iterações em abril de 2026. Nome evoluiu de Hype Station para ASN Supply.

## Evolução do Projeto
1. **Início:** React
2. **Migração:** PHP
3. **Final:** HTML puro (maior simplicidade e controle)

## Funcionalidades Implementadas

### Loja
- Catálogo de produtos (roupas e tênis)
- Sistema de favoritos
- Provador digital
- SAC widget no canto da tela

### Entrega
- Cálculo de frete por CEP
- Delivery via metrô SP — R$15 fixo

### Painel Admin
- Gestão de produtos
- Gráficos financeiros com Chart.js

### Branding
- Logo imersiva fullscreen
- Identidade visual Hype Station → ASN Supply

## Stack Técnica
- HTML, CSS, JavaScript puro
- Chart.js para gráficos financeiros
- Cálculo de frete via CEP (API)

## Decisões Tomadas
- Migrou de React/PHP para HTML puro pela simplicidade
- Delivery por metrô SP como diferencial logístico local
- Rebranding de Hype Station para ASN Supply
