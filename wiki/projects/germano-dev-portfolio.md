# Germano Dev — Portfólio Profissional

## Objetivo
Portfólio profissional com dashboards interativos de resultados para clientes, sistema de administração e apresentação visual de projetos.

## Status
**Ativo** — em evolução contínua.

## URL
germanodev-portfolio.netlify.app

## Funcionalidades

### Sistema Admin
- Acesso via duplo clique no logo "GERMANO DEV."
- CRUD completo de projetos
- Editor lateral em tempo real
- Dados persistidos em localStorage

### Dashboards de Clientes
Resultados antes/depois para 3 clientes com gráficos Chart.js:
- **Pringles** — métricas de campanha
- **Alma Estudos Visuais** — engajamento
- **Agit's Academia** — performance

### Seção de Cursos
- Gerenciamento de cursos no painel admin
- FAB speed dial para adicionar
- localStorage (`gd_courses_v1`)

### PDF do Portfólio
- Versão PDF profissional para envio
- Projetos: Pringles, TypeOn, Alma, Agit's, BolsoPro

## Stack Técnica
- HTML, CSS, JavaScript puro
- Chart.js para gráficos
- localStorage para persistência
- Netlify para deploy
