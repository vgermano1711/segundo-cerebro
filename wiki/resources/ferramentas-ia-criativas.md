# Ferramentas de IA Criativas

Ferramentas de IA que Victor usa além do Claude — para design, vídeo, apresentações e automação.

---

## Design e Imagem

### Krea.AI
- Geração e edição de imagens com IA
- Bom para criar assets visuais para projetos

### Runway ML
- Geração de vídeo com IA
- Edição e efeitos visuais automatizados

---

## Apresentações e Ideação

### Gamma
- Criação de apresentações com IA
- Victor usa para criar decks rapidamente a partir de prompts
- Usado no projeto portfólio Germano Dev

### IdeaMap.AI
- Mapas mentais gerados por IA
- Útil para brainstorming e estruturação de ideias

---

## Desenvolvimento e Prototipagem

### Bolt.AI
- Prototipagem rápida de apps com IA
- Gera código funcional a partir de descrições

### Replit
- IDE online com IA integrada
- Bom para testar ideias rapidamente sem configurar ambiente

---

## Automação

### N8N
- Automação de workflows no-code/low-code
- Integração entre ferramentas e APIs
- Victor usa para automatizar processos

### Claude Code / Cowork
- Desenvolvimento assistido por IA
- Ferramenta principal de Victor para projetos web
