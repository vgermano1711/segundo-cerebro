# 🧠 Segundo Cérebro — Obsidian + Claude AI

> Setup pessoal de produtividade combinando Obsidian e Claude AI para captura e gestão de conhecimento.

---

## 📌 Sobre o Projeto

Sistema de **Segundo Cérebro** inspirado na metodologia *Building a Second Brain* (Tiago Forte), integrando Obsidian com Claude AI para processar, resumir e expandir o conhecimento acumulado.

---

## 🎯 O que este setup permite

- 📝 Capturar e organizar anotações de forma estruturada
- 🔗 Criar conexões entre ideias com links bidirecionais
- 🤖 Usar o Claude AI para resumir, expandir e questionar notas
- ⚙️ Automatizar tarefas com scripts Batch (Windows)

---

## 🛠️ Tecnologias & Ferramentas

| Ferramenta | Uso |
|-----------|-----|
| Obsidian | Base de conhecimento pessoal (notas em Markdown) |
| Claude AI | Assistente de IA para processamento de conhecimento |
| Batchfile (.bat) | Scripts de automação no Windows |

---

## 📁 Estrutura do Repositório

```
segundo-cerebro/
├── abrir-claude.bat     # Abre o Claude integrado ao vault do Obsidian
├── sincronizar.bat      # Sincronização automática das notas
├── AGENTS.md            # Configuração dos agentes de IA
├── CLAUDE.md            # Instruções e contexto para o Claude AI
├── sources/             # Fontes e referências externas
└── wiki/                # Base de conhecimento organizada
```

---

## 💡 Inspiração

Baseado na metodologia **PARA** (Projects, Areas, Resources, Archive) e no conceito de **Zettelkasten** para conexão de ideias.
