# 🧠 Segundo Cérebro — Obsidian + Claude AI

Setup completo de **produtividade e gestão de conhecimento** combinando o poder do **Obsidian** com a inteligência do **Claude AI**.

---

## 📌 Sobre o Projeto

Sistema de **Segundo Cérebro** — metodologia de organização do conhecimento pessoal inspirada no conceito de *Building a Second Brain* (Tiago Forte), integrando Obsidian com Claude AI.

---

## 🎯 O que este setup permite

- 📝 Capturar e organizar anotações de forma estruturada
- 🔗 Criar conexões entre ideias com links bidirecionais
- 🤖 Usar o Claude AI para resumir, expandir e questionar notas
- ⚙️ Automatizar tarefas com scripts Batch (Windows)

---

## 🛠️ Tecnologias & Ferramentas

| Ferramenta | Uso |
|-----------|-----|
| Obsidian | Base de conhecimento pessoal (notas em Markdown) |
| Claude AI | Assistente de IA para processamento de conhecimento |
| Batchfile (.bat) | Scripts de automação no Windows |

---

## 💡 Inspiração

Baseado na metodologia **PARA** (Projects, Areas, Resources, Archive) e no conceito de **Zettelkasten** para conexão de ideias.
